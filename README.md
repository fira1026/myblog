# README

This is my first time to build a blog by Rails, and this blog is for demo purpose.


### Day 1

1. 設定 Rails 運行環境
2. 建立 myblog 空專案，將 myblog 推至 github
3. 將 myblog 布署至 Heroku

### Day 2

1. 建立 models: Author, Post, Comment
2. 建立並載入 fixtures
3. 新增首頁及 About 頁面
